# Desafio – HTML5 e CSS3 (Módulo 2 – Curso em Vídeo)

Este projeto foi desenvolvido como parte do curso **HTML5 e CSS3**, ministrado por Gustavo Guanabara (Curso em Vídeo).  
O objetivo do desafio foi aplicar na prática os conhecimentos estudados nos módulos 1 e 2, com foco em estruturação semântica, organização de código e estilização com CSS3.

---

## Objetivo do Projeto

O desafio consistiu em construir uma página informativa sobre a história do mascote do Android, aplicando:

- HTML semântico  
- Estruturação adequada de conteúdo  
- Uso de imagens adaptáveis  
- Responsividade básica  
- Estilização visual com CSS3  
- Organização do layout  

---

## Recursos Utilizados

### HTML5
Foram utilizadas diversas tags semânticas para melhorar a compreensão estrutural da página, tais como:

- `header`
- `nav`
- `main`
- `article`
- `aside`
- `footer`
- `figure` e `picture` para imagens responsivas

### CSS3
No estilo, foram aplicados:

- Importação e uso de fontes personalizadas
- Gradientes
- Hover em menus e links
- Responsividade para imagens
- Uso de variáveis CSS (`:root`)
- Organização do layout com espaçamentos e alinhamentos

---

## Estrutura de Pastas

---

## Como Executar

1. Baixe ou clone este repositório.  
2. Abra o arquivo `android.html` em qualquer navegador moderno.  

Não é necessário servidor local ou ferramentas adicionais.
